## 表B · 个股财务（c.css，按报告期）


| 描述变量    | 需要的数据                     | 占位字段             | PIT注意                                                                                      |
| ----------- | ------------------------------ | -------------------- | -------------------------------------------------------------------------------------------- |
| DTOA        | 资产负债率                     | `LIBILITYTOASSET`    | 用截至取数日已公告的最新报告期                                                               |
| VSAL / SGRO | 营业收入（近5年年度）          | `INCOMESTATEMENT_9`  | 建议直接取年报营业总收入                                                                     |
| VERN / EGRO | 归母净利润（近5年年度）/ EPS   | `INCOMESTATEMENT_60` | EGRO 用 EPS 序列回归                                                                         |
| ATO         | 总资产周转率(TTM)              | `ASSETTURNRATIOTTM`  |                                                                                              |
| ROA         | 总资产收益率(TTM)              | `ROA`                | 息税前利润\*2/（期初总资产+期末总资产)\*100%确认 TTM 口径字段，而且这个叫总资产报酬率（ROA） |
| GPM         | 毛利率(TTM)                    | `GPMARGIN`           |                                                                                              |
| BTOP        | 每股净资产 BPS（或净资产总额） | `BPS`                | BTOP = BPS/股价 或 净资产/总市值                                                             |
| ETOP        | EPS(TTM)（或净利润TTM）        | `EPSTTM`             | ETOP = EPSTTM/股价                                                                           |
| DTOP        | 近12月股息率                   | `DIVIDENDYIELDY`     | 这个也改了                                                                                   |

**PIT 关键确认**：c.css 财务取数是否支持「报告期 + 截止公告日」双参数（避免未来函数）。若不支持，需额外拉「定期报告公告日期」字段，自行做 point-in-time 对齐。
