# 04 · A股多因子 ML 选股框架

> 从零搭一套 A股多因子研究流水线——挖 **450+ 因子**、用 **LightGBM** 合成、做组合优化,最后**亲手用实验证明自己的 alpha 主体是微盘溢价、不可扩容**,诚实交付三个可复现策略。
>
> **这个项目的价值不在夏普,在研究规范和诚实。** 完整叙事见 [`产出/README.md`](产出/README.md);策略详情见 [`docs/最终三个策略.md`](docs/最终三个策略.md)。

## 定位
量化私募筛人看的不是"回测夏普多高"(谁都能调好看),而是:① **不作弊**(防前视)、② **工程能力**、③ **科学方法**、④ **诚实**(知道自己数字的水分)。全项目围绕这四点。

## 研究流水线
```
数据(CSMAR价量/财报/微观结构/另类 + iFinD真实中证1000成分)
  → L0引擎(去极值→中性化→标准化→IC/分层, 防前视测试全过)
  → 因子库450+(价量/GTJA191公式解释器/遗传规划/微观结构/基本面/事件/另类)
  → L3 ML合成(231因子 → LightGBM purged&embargoed walk-forward → 样本外RankIC 0.107)
  → L4/L5(扣成本回测 / Barra风险模型 / cvxpy优化 / 市场中性 / 指数增强)
  → 诚实检验(归因纠偏 / 身份实验 / CTA自我证伪 / 抗过拟合)
```

## 研究经过(思路演进 + 关键诚实发现)
| 阶段 | 诚实发现 |
|---|---|
| 因子挖掘 | 450+公式全坍缩到"流动性·反转·低波·规模"几个 premia → **novelty 来自新数据不是新数学** |
| ML合成 | LightGBM≈岭回归 → 价值在加权不在非线性;深度学习(AlphaNet)撞同一天花板 |
| 救回撤 | 质量筛选无效(回撤是结构性beta);市场中性有效但**贴水贵(约10%)** |
| 归因纠偏 | "2/3是alpha"是**中证500基准的错觉**;对真实小盘基准,回撤主体是 size beta |
| CTA证伪 | "加CTA免费午餐"经**剔除最佳月**检验后**撤回**——增益压在 2024-01 一个月=过拟合 |
| **身份实验** | 同套 alpha 的 RankIC 全市场 0.107→中证1000域 0.056→中证800域 0.030→大盘 **0.002**,**edge 是微盘溢价、不可扩容** |
| 抗过拟合定稿 | 分行业中性是真稳健改进;CDaR 只是"用收益换回撤"的拨盘、对指增有害 |

## 三个最终策略(样本外 2018-07~2025,扣换手)
| 策略 | 年化 | 夏普 | 最大回撤 | 卡玛 | 超额/IR | 定位 |
|---|---|---|---|---|---|---|
| ① naive 微盘多头 | 22.3% | 0.93 | −25.6% | 0.87 | +18%/1.10 | 收益最高 |
| ② 分行业中性微盘多头 | 20.4% | 0.91 | −22.4% | **0.91** | +16%/1.07 | 卡玛最高·去行业赌注 |
| ③ 中证1000指增 | 13.6% | 0.58 | −28.5%(绝对) | 0.48 | +8%/**1.18** | 可规模化·超额回撤−10% |

## 局限与下一步(诚实)
- **微盘策略容量受限、回撤 −22~26% 目前上不了实盘**——微盘 beta 的结构性代价,场内无干净对冲。
- **下一步**:用**杠杆/拥挤另类数据**(两融/微盘拥挤度/IM贴水/北向)做早预警 de-risk;前提是**延长历史到 2015-2016** 补够崩盘样本,否则是过拟合。
