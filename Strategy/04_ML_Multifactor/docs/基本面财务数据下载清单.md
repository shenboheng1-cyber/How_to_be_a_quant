# 基本面财务数据下载清单（CSMAR）

> 和另类数据并列的另一条腿。**质量/低应计因子正好治"多头持垃圾股、2024 小盘崩盘"那条病**——你做空不了垃圾股，那就用基本面**不持有**它们。
> 配套：[另类数据因子完整目录](另类数据因子完整目录.md) · [价量/高频清单](下一步数据下载清单.md)

## 战略定位
- **基本面 = 慢、稳、抗崩盘的腿**：季频、低换手，本身 IC 不一定高，但**与价量正交、改善组合 IR 与回撤**。
- **质量(Quality) 是对你项目最对症的一类**：ROE/毛利率/低应计/低资产扩张 → 把组合从"小盘垃圾"偏向"赚钱的好公司"。
- **⚠️ PIT 铁律**：财务数据**必须用「实际公告日」对齐**（`IAR_Rept`），不能用报告期 `Accper`，否则前视。
- **TTM 处理**：净利润/营收/现金流等**流量**项要算**滚动 12 个月(TTM)**，避免季节性。

## 因子家族 → CSMAR 来源（要写全）

| 因子家族 | 标准因子 | 用哪张表 | A股有效性 |
|---|---|---|---|
| **价值 Value** | EP=净利TTM/市值、BP=净资产/市值、SP=营收TTM/市值、CFP=CFO/市值 | `FS_Comins`(净利/营收)+`FS_Combas`(净资产)+`FS_Comscfd`(CFO)，**除以你现有的当前市值** | 中等(BP/EP 显著但弱于美股) |
| **质量 Quality** ⭐ | ROE、ROA、ROIC、销售毛利率、净利率 | `FI_T5`(盈利能力，**现成比率**) | **好**——避雷垃圾股的核心 |
| **成长 Growth** | 营收/净利/净资产 同比增速 | `FI_T8`(发展能力) | 混合(高成长常被高估) |
| **投资 Investment** ⭐ | 总资产增长率(**负向**，资产扩张异象) | `FI_T8` 总资产增长率 | **好**(资产猛扩张→未来跑输) |
| **盈余质量/应计** ⭐ | 应计=(净利−CFO)/总资产(**负向**，Sloan) | `FS_Comins`+`FS_Comscfd`+`FS_Combas`；或现成 `AIQ_*`(见另类目录) | **好**——避雷核心 |
| **现金流质量** | CFO/营收、CFO/净利(盈余含金量)、自由现金流 | `FI_T6`(现金流分析)+`FS_Comscfd` | 中等偏好 |
| **杠杆/偿债** | 资产负债率、流动比、速动比、利息保障倍数 | `FI_T1`(偿债能力) | 风险因子(高杠杆→风险) |
| **营运效率** | 总资产周转、存货周转、应收周转 | `FI_T4`(经营能力) | 弱-中(与质量相关) |
| **每股指标** | EPS、每股净资产、每股CFO、每股未分配利润 | `FI_T9`(每股指标) | 配合价值用 |
| **分红 Dividend** | 股息率、分红率、分红连续性 | `FI_T11`(股利分配)/`CD_Dividend` | 中等(高股息防御) |
| **财务风险** | 综合财务风险指标 | `FI_T7`(风险水平) | 风险/避雷 |

> ⭐ = 对症"长线多头持垃圾股"问题、最该优先做的几类。

## 两条实现路径

1. **省事路径（推荐先用）**：直接下 **`FI_T1`~`FI_T11`**（财务指标分析，CSMAR 已算好全部比率）→ ROE/毛利率/增速/杠杆/周转/每股 全有，最快出质量/成长/杠杆因子。
2. **原始路径（做自定义因子）**：下 **`FS_Combas`/`FS_Comins`/`FS_Comscfd`** 三大原始报表 → 做**用当前市值的价值因子**、**应计**、**自由现金流**等比率表里没有或口径要自定的因子。

## 完整下载清单

> 设置：CSV、字段全选、时间 2010 起（财务可拉更早 2007）、丢进 `data_before_clean/`。

### 🥇 核心必下
| 数据库 | 表 | 用途 |
|---|---|---|
| **财务指标分析** | `FI_T1`,`FI_T4`,`FI_T5`,`FI_T6`,`FI_T8`,`FI_T9`（或干脆全下 T1–T11，很小） | 质量/成长/杠杆/周转/现金流/每股 现成比率 |
| **财务报表** | `FS_Combas`(资产负债表)、`FS_Comins`(利润表)、`FS_Comscfd`(现金流量表-直接法) | 价值(配当前市值)、应计、自定义 |
| **年、中、季报公布日期** | `IAR_Rept`(实际公告日) | **PIT 对齐必下** |
| **红利分配** | `CD_Dividend` | 股息率/分红 |

### 🥈 可选/进阶
- `FS_Comscfi`（现金流量表-间接法）：某些应计口径要用。
- `FI_T2`,`FI_T3`,`FI_T7`,`FI_T10`,`FI_T11`：补全披露指标/比率结构/风险/相对价值/股利。
- **银行/证券/保险三表** `FS_BCombas`/`FS_SCombas`/`FS_ICombas` 等：**仅当你要把金融股纳入**时下；否则**金融股单独处理或直接剔除**（它们报表结构不同，普通因子不可比）。
- `财务报表附注` `FN_*`(90张)：做研发资本化、减值、明细类高级因子时按需，**别一次全下**。

## 诚实提醒（坑）
- **频率季频**：因子季度才更新一次，换手低——这是优点(抗成本)，但要和高频价量因子**分层合成**。
- **金融股**：银行/券商/保险三表不同，普通财务因子对它们不可比，**标准做法是剔除金融股**或单独建模。
- **PIT**：务必 `IAR_Rept` 公告日对齐；季报通常滞后 1–4 个月，年报集中在次年 4 月底。
- **A股有效性现实**：价值中等、质量好、成长混合、**资产增长(负向)和应计(负向)是验证最稳的两个**——也正好是避雷信号。

## 下载顺序建议
1. **`FI_T5`(盈利) + `FI_T8`(发展) + `FS_Combas/Comins/Comscfd` + `IAR_Rept`** ← 一次拿下 质量+成长+价值+应计+PIT
2. `FI_T1/T4/T6/T9` + `CD_Dividend` ← 杠杆/周转/现金流/每股/分红
3. 其余按需
